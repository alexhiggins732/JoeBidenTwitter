- 3,731 accounts were just created.
- 2,871 have zero followers.

The file [JoeBiden.json.zip](https://github.com/alexhiggins732/JoeBidenTwitter/blob/main/joebiden.json.zip "JoeBiden.json.zip") is a compressed zip file of the Json of the accounts Joe Biden's latest followers.
